# Robert-96.github.io

My boring personal website.

## Development Setup

### Prerequisites

* [hugo](https://gohugo.io/)

### Start the Hugo server

```bash
$ hugo serve -D
```

Navigate to the site at http://localhost:1313/.

### Build

```bash
$ hugo -D
```

Output will be in `./public/` directory by default (`-d/--destination` flag to change it).
